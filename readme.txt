*TO PUSH CHANGES

git status
*compare local files to repo on github
git commit -a
*stage changes with comment
[write the comment] ESCAPE button :wq + ENTER button
git push origin main
*push everything that is staged to the main
clear
*clear the screen of text history
git branch <branch name>
*creates new branch
git checkout <branch name>
*switch to the targetted branch
git branch -d <branch name>
*delete targetted branch
